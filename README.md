# MagicStick

## サルでもわかる使い方
### 1. 「OP」になる。
### 2. 「木の棒」を持つ。
### 3. 使用者が「KUN」である。
### 4. コマンドでモードを選択する。
### 5. 対象に向かって左クリックする。

### コマンド
#### /stick ban : 赤い光線になる。光に当たった者をBANする。
#### /stick kill : 緑の光線になる。光に当たった者をLILLする。
#### /stick sky : 青い光線になる。光に当たった者を10万ブロック上空に吹っ飛ばす。
#### /stick off : ただの木の棒になります。保護機能です。
